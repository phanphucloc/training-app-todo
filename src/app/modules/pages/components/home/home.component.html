Todo app demo


Description

  • This is a small web signal page built by framework angular version 10.0.1

Features

  • Application can change language (default language is English)
  • Provide basic features add, delete, edit
  • Features filtering data using Observables & Rxjs
  • Using akita to manage State management
  • Implement reactive form, create abstract form for 2 both of cases (creation, edition)

Packpage

  • @ng-bootstrap/ng-bootstrap - version ^6.1.0
  • @datorama/akita - version ^5.2.1
  • @angular/material - version ^10.1.2
  • rxjs - version 6.5.5
  • typescript - version 3.9.5
